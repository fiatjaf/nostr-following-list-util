# nostr-following-list-util (NostrFlu)

[Japanese](READMEja.md)

## Features

- Find and display past `kind:3` (Contacts) events from multiple relays
- Export following lists
- Create and send a new event using selected/imported lists

## Dependencies

- Vite
- Preact
- nostr-tools
